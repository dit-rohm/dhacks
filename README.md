## Requirements

- npm
- compass

## Getting Started

Install dependencies:

```
$ npm install
$ npm run bower install
```

Run the local server:

```
$ npm run start
```

## Build

```
$ npm run build
```

## Deploy

```
$ npm run deploy
```
